Code Runner
===========

An application to run code

Installation:
=============
- `brew install boot2docker`
- added runner to path

Usage:
=====
    runner [-hl] [FILE]
    Run code and edit it in the same window...
    
        -h          display this help and exit
        -l LANGUAGE Language to use

Languages Supported:
====================

- Go: -l golang
- Node.js: -l node
